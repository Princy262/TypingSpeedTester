# Typing Speed Test

## Overview
The Typing Speed Test is a web application designed to help users improve their typing speed and accuracy. It presents users with random paragraphs to type within a set time limit, tracking their performance and providing feedback on their typing accuracy.

## Files
- **js/property.js**: Contains the main JavaScript logic for the application, including functions for generating paragraphs, initializing typing, managing the timer, and resetting the game.
- **index.html**: The main HTML document that sets up the structure of the web page, including input fields and display areas for typing statistics.
- **style.css**: Contains styles for the application, defining the visual appearance of HTML elements, including layout, colors, and fonts.

## Setup
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to run the application.

## Usage
- Click on the paragraph or press any key to focus on the input field.
- Start typing the displayed text within the time limit.
- The application will track your typing speed (WPM) and mistakes.
- Click the "Try Again" button to reset the game and generate a new paragraph.
- Toggle between dark mode and light mode using the switch provided on the interface for a comfortable viewing experience.
